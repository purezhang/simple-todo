# Simple Todo

基于 C++/WTL 的极简任务管理器

## 软件架构

### 核心组件
- **MainFrm** - 主框架窗口，负责整体布局和命令处理
- **TodoListCtrl** - 任务列表控件，使用 Virtual List-View 实现高性能显示
- **TodoDataManager** - 数据管理类，处理任务的增删改查
- **SQLiteManager** - 数据库管理类，负责数据持久化

### 技术栈
- **C++11** - 核心开发语言
- **WTL (Windows Template Library)** - GUI 框架
- **SQLite** - 数据持久化存储
- **Visual Studio** - 开发环境

### 主要功能
- Virtual List-View 高性能显示
- 按日期分组，支持折叠/展开
- 优先级颜色标识
- SQLite 数据持久化
- 支持 todo.txt 格式的导出
- 支持 csv 格式导出

## 构建说明

### 开发环境
- Visual Studio 2022 或更高版本
- Windows 10/11 操作系统

### 构建步骤
1. 克隆代码仓库
2. 使用 Visual Studio 打开 SimpleTodo.sln 解决方案
3. 选择 Debug 或 Release 配置
4. 编译并运行项目

### 依赖项
- SQLite 3.40.0+
- WTL 10.0+

## 使用说明

1. **添加任务**：点击工具栏的"添加"按钮，或使用快捷键 Ctrl+N
2. **编辑任务**：双击任务项，或右键选择"编辑"
3. **删除任务**：右键选择"删除"
4. **标记完成**：选择任务后，点击工具栏的"完成"按钮
5. **导出任务**：点击"文件"菜单，选择"导出"选项
6. **调整优先级**：右键选择"优先级"子菜单

## 常见问题

### 任务删除后无法恢复
- 任务删除后会立即从数据库中删除，无法恢复，请谨慎操作。

### 导出格式说明
- **todo.txt**：遵循标准的 todo.txt 格式，包含任务标题、优先级和完成状态
- **CSV**：包含任务的所有字段，可用于在其他应用中导入和编辑

### 数据存储位置
- 数据存储在 `data/simpletodo.db` 文件中
