# Issue-17: 叠加筛选集成（在已完成列表中筛选）

## 需求描述

将所有筛选功能（置顶、时间、搜索、项目）集成，确保在 TodoList 和 DoneList 中都能正常使用叠加筛选。

## 功能规格

### 叠加筛选逻辑

所有筛选条件同时生效：
```
显示任务 = 置顶筛选 AND 时间筛选 AND 搜索过滤 AND 项目筛选
```

### 筛选优先级

1. **置顶** - 只显示置顶/非置顶
2. **时间** - 今天/本周/全部
3. **搜索** - 关键词匹配
4. **项目** - 按项目分组筛选

### 已完成列表

- 所有筛选条件同样适用于 DoneList
- 置顶筛选在 DoneList 中不生效（已完成任务无置顶概念）

## 实现步骤

### 步骤1：统一筛选接口

**`TodoDataManager`**
```cpp
std::vector<int> Filter(bool isDoneList, const FilterState& state);
```

### 步骤2：同步筛选状态

**`MainFrm`**
- TodoList 和 DoneList 共用同一套筛选状态
- 切换列表时保持筛选状态

### 步骤3：UI 更新

- 筛选状态变化时刷新两个列表
- 显示当前筛选条件（可选）

## 预期效果

1. 筛选条件叠加生效
2. 切换列表时保持筛选状态
3. 性能优化（避免重复筛选）
