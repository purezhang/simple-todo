# Issue-06: 实现弹出层详情面板

## 需求描述

点击任务时，在主窗口右侧弹出一个详情面板，显示任务详情。点击空白处关闭面板，点击其他任务时更新面板内容。

## 实现状态

**已完成** - 实际实现与原设计略有不同

## 实现差异

| 原设计 | 实际实现 |
|--------|----------|
| 独立窗口类 `CDetailPopupWindow` | 使用 `WS_POPUP` 样式的子窗口，直接挂载到主窗口 |
| 需要处理 WM_MOVE 跟随主窗口 | 已实现跟随主窗口移动和大小调整 |
| 使用 `CStatic` 显示文字 | 改为只读 `CEdit` 支持文字选择复制 |
| 有分割线 | 移除分割线，保留凹陷边框效果 |

## 最终布局

```
┌─────────────────────────────────┐
│           TodoList (60%)          │
├─────────────────────────────────┤
│  DoneList (40%)                  │
│         ┌──────────────┐        │
│         │   DetailPanel │ 300px  │  ← 弹出层（覆盖在 DoneList 上）
│         │   (浮层)     │        │
└─────────────────────────────────┘
```

## 功能特性

1. 点击任务 → 右侧出现 300px 宽详情面板
2. 点击其他任务 → 面板内容更新
3. 点击列表空白处 → 面板关闭
4. 拖动/调整主窗口 → 面板跟随移动/调整位置
5. 面板内文字可选择复制（只读 Edit）
6. 保持上下分割布局结构不变

## 修改的文件

- `src/MainFrm.h` - 添加成员变量，移除分割线声明
- `src/MainFrm.cpp` - 实现 popup 逻辑，添加 WM_MOVE 处理

## 备注

- 原设计中"面板固定在主窗口右侧，不随主窗口拖动而移动"已改为跟随移动
- 改为只读 Edit 替代 Static 以支持文字选择复制
