# Issue-14: 搜索过滤功能

## 需求描述

实现任务列表的实时搜索过滤功能。

## UI 设计

```
🔍________        ← 输入搜索关键词
```

## 功能规格

### 搜索优先级

搜索结果按以下优先级排序：
1. **任务描述** - 完全匹配优先
2. **截止时间** - 包含匹配
3. **创建时间** - 包含匹配

### 交互设计

1. **输入时**：
   - 停止输入后 500ms 自动开始搜索
   - 避免频繁搜索

2. **搜索条件**：
   - 不区分大小写
   - 支持部分匹配

3. **空输入**：
   - 显示全部任务

### 步骤1：添加搜索控件

### 步骤2：实现搜索逻辑

**`src/TodoDataManager.h/cpp`**

std::vector<int> Search(const std::wstring& keyword);

### 步骤3：集成到列表显示

**`src/TodoListCtrl.cpp`**

1. 输入关键词后自动搜索
2. 实时过滤任务列表
