# Issue-15: 项目筛选功能

## 需求描述

实现任务列表的项目分组筛选功能，项目筛选靠工具栏右侧显示。

## UI 设计

```
工具栏布局：
┌─────────────────────────────────────────────────────┐
│ 📌置顶 │ 🏷全部 │ 🔍________           ↓ [全部 ▼] │
└─────────────────────────────────────────────────────┘
                              ↑
                          项目筛选（居右）
```

## 功能规格

### 筛选选项

1. **全部** - 显示所有任务
2. **内置分组** - "工作"、"个人"等（可配置）
3. **自定义分组** - 用户创建的项目

### 交互方式

1. **点击轮询**（时间筛选）：
   - 🏷-今天 → 🏷-本周 → 🏷-全部
   - 按截止时间筛选

2. **下拉选择**（项目筛选）：
   - 显示分组列表
   - 居右显示

### 叠加筛选

支持多个条件同时生效：
- 置顶筛选 + 时间筛选 + 搜索 + 项目筛选

## 实现步骤

### 步骤1：添加筛选状态

```cpp
enum class TimeFilter { Today, ThisWeek, All };

struct FilterState {

    TimeFilter timeFilter;   // 时间筛选

    std::wstring keyword;   // 搜索关键词
};
```
