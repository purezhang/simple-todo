# Issue-16: 存档分组功能

## 需求描述

实现任务的存档/取消存档功能，将任务标记为"存档"状态。

## 功能规格

### 内置分组

系统内置两个特殊分组：
- **全部** - 显示所有非存档任务
- **存档** - 只显示已存档任务

### 交互方式

1. **存档任务**：
   - 右键菜单增加"存档"选项
   - 或拖拽到"存档"分组

2. **取消存档**：
   - 在存档列表中右键选择"取消存档"

3. **新建任务**：
   - 不显示"存档"选项
   - 默认不属于存档

## 实现步骤

### 步骤1：修改数据模型

**`TodoItem` 结构体**
```cpp
bool isArchived;  // 是否已存档
```

### 步骤2：修改数据库

**`SQLiteManager.cpp`**
- 添加 `is_archived` 列
- 更新 CRUD 操作

### 步骤3：添加 UI

**`MainFrm.cpp`**
- 右键菜单增加"存档"/"取消存档"
- 存档列表显示逻辑

## 预期效果

1. 可以将任务标记为存档
2. 存档任务在"存档"分组中显示
3. 可以从存档中恢复任务
